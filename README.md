# Roselle Studio — Portfolio & Creative Site

A cinematic portfolio for photographers, videographers, and digital artists — powered by Sanity CMS and Framer Motion.

## Stack
Next.js 14 (App Router) + TypeScript · Sanity v3 · Framer Motion · Resend · Vercel

## What's Included
- Fullscreen hero with showreel video or dramatic still
- Masonry project grid with hover overlays and page transitions
- Project detail pages with gallery, client info, and OG image generation
- About page with CV download and "Currently Booking" availability toggle
- Contact form via Resend

- Magnetic cursor effect (desktop), reduced-motion support




## Deployment
- **App + Studio** → Vercel
- Add production domain to Sanity CORS origins at manage.sanity.io
